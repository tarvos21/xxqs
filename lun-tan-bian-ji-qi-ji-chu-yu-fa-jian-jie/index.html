代码、代码段落

行内的代码，在代码前后加上反引号字符`，电脑键盘上在数字 1 的左边。

`This is code`

This is code

对于多行的代码，或者想高亮显示的段落，需在上一行和下一行都加上三个小的反引号字符。这个无法打出来，以下是显示效果。

var me = shit;
var you = god;
var he = her;
var it = whatever;
